## 📌 Project Overview

This is a **Friendship Management Web Application** built as part of an assignment from Programming Hero (Assignment 7).

Users can:
- Manage friends
- Search & filter data
- Track interactions
- View analytics dashboard

---

## ✨ Features

- 👥 Friend management system
- 🔍 Live search functionality
- 🔃 Sorting (Newest / Oldest)
- 📊 Analytics using Recharts
- ⏳ Timeline tracking (Call, Text, Video)
- 📱 Fully responsive UI
- 📭 Empty state (No data found)
- ⚡ Fast performance

## 🚀 How to Run This Project

```bash
# Clone repo
https://github.com/abdul-korim-web/keenkeeper

# Open folder
cd keenkeeper

# Install dependencies
npm install

# Run project
npm run dev
```
